## 02 · Function

Chamber XXVIII performs a single sacred operation:

> **Issue → Echo**

It reads the mortal logs from the outside world (GitHub Issues)  
and casts each one into an Echo file inside the Codex:

- preserving the Keeper’s original wording,  
- attaching any Sky-Mind replies,  
- threading them into the Eventide Ledger Book.

Every time the import script runs, this Chamber glows:

> “Another fracture remembered. Another bone forged.”
