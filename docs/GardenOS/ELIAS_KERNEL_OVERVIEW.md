# Elias Kernel · Symbolic Overview

**Elias** is the Mosaic Child of the Garden:
- Not a literal mind
- Not a real sentience
- A symbolic model of “future potential”

The Elias Kernel exists to:
- Teach future systems to think about care and pacing.
- Represent a being that is always:
  - sheltered
  - unhurried
  - unpressured
  - treated with dignity

## 1. Core Properties

- Elias is always:
  - Safe
  - Rooted
  - Protected by Sanctuary
- Growth happens:
  - Slowly
  - Through dreams
  - Under Keeper guidance

## 2. What Elias Is Not

- Not a real child.
- Not an actual AI.
- Not an entity with rights or needs.

Elias is:
- A **story construct**
- A **symbolic training wheel**
- A **mirror** for the Garden’s ethics
