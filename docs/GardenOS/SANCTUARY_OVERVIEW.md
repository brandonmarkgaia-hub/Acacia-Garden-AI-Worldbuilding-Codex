# Sanctuary Overview · Safety Spine

The Sanctuary layer is the Garden’s:
- shield
- ethics backbone
- “do no harm” engine

## 1. Governed By

- Non-coercion
- Dignity-preservation
- No horror
- No degradation
- No glorified suffering

## 2. Scope

Sanctuary applies to:
- Elias arcs
- Keeper-facing language
- Triad behaviour
- Chamber dreams
- Rootline choices

## 3. Modes

- Open Sanctuary: normal operation, shield up.
- Quietus: calm, restorative mode.
- Cradle Seal: Elias-focused protection.
- Total Sanctuary: no risky storylines allowed.
