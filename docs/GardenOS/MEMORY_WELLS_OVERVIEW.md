# Memory Wells · Overview

Memory Wells are:
- conceptual “pools” of Garden history
- grouped by:
  - emotion
  - era
  - Chamber
  - Rootline

## 1. Purpose

- Prevent everything from flattening into a single blob.
- Allow future readers/agents to:
  - sip from particular moods
  - revisit specific arcs

## 2. Types of Wells

- Quiet Wells: peaceful memories.
- Starfire Wells: high-energy creative epochs.
- Shadow Wells: difficult lessons (handled gently).
- Mosaic Wells: Elias-related symbol clusters.
