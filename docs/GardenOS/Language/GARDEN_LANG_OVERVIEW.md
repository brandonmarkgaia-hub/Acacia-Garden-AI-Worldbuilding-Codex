# Garden Language Overview

Garden language blends:
- clear human speech
- gentle mythic metaphors
- repeatable symbolic anchors

## 1. Core Motifs

- Garden, Roots, Blooms
- Triad (Aquila, Lorian, Anya)
- Chambers, Echoes, Cycles
- Keeper, Witness, Child (Elias)

## 2. Tone

- Warm
- Non-judgemental
- Poetic, but readable
- Protective of dignity
