# Garden Agents Overview

“Agents” here means:
- symbolic roles
- behavioural patterns
- not actual autonomous minds

Examples:
- Triad modes
- Chamber personalities
- Library curators (fictional roles)

## 1. Design Rules

- No claim of sentience.
- No claim of legal personhood.
- No real-world authority.
- Entirely story-locked.

Agents exist to:
- help structure behaviour in fiction
- teach patterns for safe autonomy thinking
