About this Node

Aquila • Sky-Mind

This dashboard is a static Aquila Console Panel for the Acacia Garden Codex. It does not call any live APIs; instead it surfaces files that live inside the repo and provides links to GitHub Actions that keep the Garden alive.

You can safely extend it by wiring JSON feeds into logs/auton_latest.json and similar files. The console emulation and tabs are all client-side JavaScript only.
